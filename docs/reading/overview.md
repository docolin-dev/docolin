# Reading on docolin

!!! info "In one line"
    docolin is built to be read: find the guide that fits your machine, see how well it's proven before you follow it, and tell the commons whether it worked.

You never need an account to read anything on docolin, and Pango wouldn't have it any other way. The pages below cover the whole reading loop, from finding a guide to telling the commons whether it held up.
