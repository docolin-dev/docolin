## Native video

An image whose URL points at a video file (`.mp4`, `.webm`, ...) renders as a real
video player, no special syntax needed:

![Big Buck Bunny](https://download.blender.org/peach/bigbuckbunny_movies/BigBuckBunny_320x180.mp4)

## YouTube, the private way

The same image form embeds a YouTube video. Only the thumbnail loads up front; no
player, cookies, or tracking reach the reader until they press play:

![Big Buck Bunny, the short film](https://www.youtube.com/watch?v=aqz-KE-bpKQ)

A plain [YouTube link](https://youtu.be/aqz-KE-bpKQ) stays a link. Only the image
form embeds.
